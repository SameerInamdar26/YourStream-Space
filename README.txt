Features

* Upload videos with title, description, and thumbnail
* Support for YouTube/iframe video links
* View all uploaded videos in one place
* Clean, modern, and responsive UI

Tech Stack

* **Frontend:** HTML, CSS, JavaScript
* **Backend:** Node.js, Express
* **Database:** MongoDB

Setup

1. Clone this repo

   ```bash
   git clone https://github.com/<your-username>/YourStreamSpace.git
   cd YourStreamSpace
   ```
2. Install dependencies

   ```bash
   npm install
   ```
3. Add your MongoDB URI in `.env`

   ```
   MONGO_URI=your_mongo_connection
   PORT=5000
   ```
4. Run

   ```bash
   npm start
   ```

Then open  http://localhost:5000
